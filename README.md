Flight Delay Prediction Using Machine Learning

Overview

This project predicts flight delays using Machine Learning techniques. The system analyzes factors such as airline, aircraft type, origin airport, destination airport, departure weekday, departure time, and arrival time to determine the likelihood of a flight being delayed.

Features

- Flight delay prediction using Random Forest algorithm
- User-friendly web interface
- Data preprocessing and feature engineering
- Model training and evaluation
- Real-time prediction based on user inputs

Technologies Used

- Python
- Machine Learning
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- JavaScript

Dataset Features

- Aircraft Type
- Origin Airport
- Destination Airport
- Departure Weekday
- Departure Time
- Arrival Time
- Airline

How to Run

1. Clone the repository.
2. Install the required dependencies.
3. Run the Python application.
4. Open the web interface in your browser.
5. Enter flight details and view the prediction results.

Machine Learning Model

The project uses the Random Forest algorithm to classify and predict flight delays based on historical flight data.
